# 世界线变动率探测仪

##介绍
演示视频（B站）：https://www.bilibili.com/video/BV1yb411T746
这个项目的开始其实就是因为喜欢命运石之门里面的世界线变动率探测仪，于是就把他做了出来放在了b站上。
然后越来越多的人想要复现这个，于是我就打算把它开源出来。
之前只是把工程放在群文件里让大家加群下载 但是人越来越多就显得很麻烦 于是就在这里开源出来。

##说明
主控使用的是 STM32F030F4P6 核心板
